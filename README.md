# MyPersonalPage

這裡是 **Chicken Five** 的個人入口網站，一個融合了生活溫度與開發技術的實驗空間。


## 🎨 關於這個空間
這個網站採用 **"Cream-style" (奶油風格)** 極簡設計，旨在提供一個舒適、無負擔的閱讀環境。透過這個空間，你可以找到：
- **Code**: 我的技術作品與開發紀錄（.NET / Blazor / Linux）。
- **Cycling**: 透過 Strava 數據分享的單車訓練與騎行軌跡。
- **Coffee**: 手沖咖啡的日常。
- **Cats**: 監督我寫程式的 6 歲主子日常。

## 🛠 技術棧 (Tech Stack)
* **Framework**: [ASP.NET Core Blazor WebAssembly](https://dotnet.microsoft.com/en-us/apps/aspnet/web-apps/blazor)
* **UI Component**: [MudBlazor](https://mudblazor.com/) - 打造磨砂玻璃質感與簡約排版。
* **CI/CD**: GitHub Actions - 自動部署至 GitHub Pages。
* **Domain**: 由 Cloudflare 託管於 [me.dev-chi.com](https://me.dev-chi.com)。
* **Architecture**: 響應式 PWA 設計，支援手機桌面安裝。

## 🚀 快速開始
如果你想在本地端運行此專案：

1.  **複製專案**
    ```bash
    git clone https://github.com/th0225/MyPersonalPage.git
    ```
2.  **執行專案**
    ```bash
    dotnet watch run
    ```
